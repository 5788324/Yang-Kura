# Yang-Kura 当前路线 - MVP-31

## 状态

版本：`0.69.0-mvp31`

MVP-31 的目标是把已经能播放的打包版继续向“日常可用播放器”推进：不新增危险文件操作，不接 SQLite / 下载器，而是收口最近播放和继续播放体验。

## 本轮完成目标

```text
真实播放 / Demo 播放
→ 写入本地播放历史

→ 继续播放优先使用最近播放
→ 再次播放同一音轨时恢复上次进度
```
