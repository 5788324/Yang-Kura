## 项目定位

Yang-Kura 是个人本地音频媒体库，不商业、不对外发布、不作为开源产品运营。主线是本地资源管理、播放、导入和后续下载/元数据扩展。

```text
React + Vite + TypeScript + Electron
Local JSON Index 优先
SQLite 后置
ASMR/RJ + 普通音乐双库
中文 UI 优先
```

## 安全边界

当前项目是个人本地项目，边界不用按企业级/公网/多人协作设计；但仍保留以下硬规则：

```text
真实文件操作必须可预览、可确认、可记录。
禁止自动覆盖。
禁止自动删除。
禁止自动清理源目录。
失败应停止或明确记录。
Renderer 不接收 absolutePath / file://。
SQLite、下载器、联网元数据、mpv 都不在当前阶段插队。
```
