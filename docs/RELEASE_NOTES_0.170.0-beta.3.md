# Yang-Kura 0.170.0 Beta 3 · 正式日用候选

平台：Windows x64。

Beta 3 将 Beta 2 发布后完成的真实资源库稳定性、单实例、独立 Profile、主题/歌单/导入页修复和播放器停滞处理带入新的 portable 与 NSIS 安装包。

## 主要变化

- U40-D：Index 读取状态统一，15 秒超时、失败重试和跨页面会话一致；旧版大集合按实际 RJ/作品目录重新分组。
- U40-D2：新增 `YANG_KURA_USER_DATA_ROOT` 独立 Profile、Electron 单实例锁、授权恢复；修复歌单 `$0`、三主题标签和导入页工程术语。
- U40-D3：HTMLAudio 元数据读取和启动各增加 10 秒超时；不支持的编码不再长期停在 `0:00 / 0:00`，会明确提示到“设置 → 播放方式”选择 mpv。
- 真实 `E:\arsm` 已验证 75 个集合、3540 条轨道、mpv 播放、Seek、字幕、单实例和连续三次重启。
- Git Fast Lane v2 生效：普通维护不再反复触发完整打包，正式 Release 才执行一次 L3 全链。

## 安装与升级

- 可使用 portable 或 NSIS setup。
- 支持覆盖安装；卸载默认保留用户数据。
- 从 Beta 2 升级前不需要迁移媒体文件或重建真实目录。
- 建议首次播放前在“设置 → 播放方式”确认 mpv；未检测到时可选择本机 `mpv.exe`。

## 数据边界

- 媒体库、Index、歌单、队列和历史均保存在本机。
- 真实媒体目录默认只读；导入写入采用预览、确认、日志和回滚边界。
- 不上传媒体文件，不包含账号、云同步或遥测系统。

## 已知限制

- 未进行商业代码签名，Windows 可能显示来源提示。
- 不包含自动更新，需要手动下载新版本。
- HTMLAudio 不能保证支持所有音频编码；失败时应选择 mpv。
- 正式下载器、SQLite 全迁移、OpenList/WebDAV、Player Core v2、完整 AI Agent、转录集成和云同步仍冻结。
