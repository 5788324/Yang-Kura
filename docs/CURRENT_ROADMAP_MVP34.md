# CURRENT ROADMAP — MVP-34 播放器体验基础精修

## 当前版本

`0.72.0-mvp34`

## 阶段目标

MVP-34 不改变扫描、索引写入、字幕读取、外部打开、打包链路。它只收口播放器体验：

```text
本地音频播放已经可用
→ 播放器界面减少演示感和工程词
→ 增加更适合 ASMR / 长音频的播放结束策略
→ 保留睡眠定时 / 睡前低亮度入口
```

## 本轮允许范围

- `PlayerState` 增加播放结束策略。
- `useAudioPlayer` 在音轨结束时应用策略。
- `PlayerBar` 显示用户可理解的播放状态 / 字幕状态 / 结束策略。
- `LyricsPanel` 保留三种播放器视图，同时增加结束策略入口。
- 清理播放器表层的假 `VIP`、假点赞数、英文工程提示和固定 Hi-Res 文案。

## 本轮禁止范围

- 不接 SQLite。
- 不接下载器。
- 不抓 ASMR.one / DLsite / 网易云元数据。
- 不删除、移动、重命名真实文件。
- 不改 `library-index.json` 写入规则。
- 不改 tokenized local-file 安全边界。
- 不引入 mpv / VLC / PotPlayer 可控后端。

## 新增用户能力

播放器现在有三种播放结束策略：

| 策略 | 行为 |
|---|---|
| 列表续播 | 当前轨结束后继续队列下一轨 |
| 播完当前轨停止 | 当前轨结束后暂停 |
| 播完当前作品停止 | 同一 RJ / 同一专辑内继续，离开当前作品时暂停 |

这比普通音乐播放器更适合 ASMR / 长音频 / 睡前使用。
