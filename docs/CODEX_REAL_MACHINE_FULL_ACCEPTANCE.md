# Codex 实机全链路验收 — Yang-Kura U40-D

## 1. 任务目标

在 Windows 实机上使用真实媒体库完成一次完整、可复核、不会污染用户数据的产品验收。

真实资源库：

```text
音声库：E:\arsm
音乐库：D:\CloudMusic\VipSongsDownload
```

验收必须覆盖：

- 全部可见页面；
- 全部可见按钮、表单、下拉框、菜单、弹窗和抽屉；
- 鼠标与键盘操作；
- 真实资源库读取与重启恢复；
- 音声作品和音乐分组；
- 播放器、队列、历史、续播、字幕和全屏歌词；
- 导入器的安全流程；
- 三套主题；
- 窗口尺寸、显示缩放和减少动画；
- 应用关闭、重新启动后的状态恢复。

本任务是实机验收，不进行功能开发。发现代码问题时记录完整证据，不在未定位根因前做临时绕过。

## 2. 绝对安全边界

### 2.1 真实资源库只读

对以下目录只允许选择、读取、扫描预览、播放、加载封面和字幕：

```text
E:\arsm
D:\CloudMusic\VipSongsDownload
```

禁止对真实资源库执行：

- 删除文件；
- 移动文件；
- 重命名文件或目录；
- 覆盖媒体文件；
- 批量改写元数据；
- 清理真实资源库；
- 修改已有字幕；
- 在没有备份的情况下改写 `library-index.json`；
- 使用导入器把文件写入真实资源库。

如果真实目录不存在 `library-index.json`，只执行“扫描预览”，不要在真实目录点击会写入记录的操作。需要验证写入链时使用第 2.2 节的临时副本。

### 2.2 导入与写入测试只使用临时目录

创建：

```text
%TEMP%\YangKura-U40D-RealAcceptance\source
%TEMP%\YangKura-U40D-RealAcceptance\target
%TEMP%\YangKura-U40D-RealAcceptance\evidence
```

从真实媒体库中只复制少量样本到 `source`：

- 2 个短音频；
- 1 张封面；
- 1 个字幕；
- 保留两层以上目录结构。

所有导入、复制、受控移动、回滚、Index 写入、备份和恢复测试都在临时目录完成。

验收结束后只删除 `%TEMP%\YangKura-U40D-RealAcceptance`，不得清理真实库。

### 2.3 配置隔离

使用单独实机验收配置目录：

```text
%LOCALAPPDATA%\Yang-Kura-U40D-RealAcceptance
```

不要复用自动化测试配置，也不要把 `U28`、`U29`、`U30`、`U31`、`U32`、`U40-B` 样本写入日常 profile。

## 4. 证据要求

证据目录：

```text
%TEMP%\YangKura-U40D-RealAcceptance\evidence
```

每个失败项至少保存：

- 页面全屏截图；
- 局部问题截图；
- 操作前后状态；
- 时间；
- 窗口尺寸；
- Windows 缩放；
- 主题；
- 复现步骤；
- 预期与实际；
- 是否重启后仍复现；
- 控制台或应用日志；
- 是否涉及真实文件写入。

缺陷编号格式：

```text
U40D-GUI-001
U40D-PLAYER-001
U40D-INDEX-001
U40D-IMPORT-001
```

严重度：

- Blocker：主链无法继续、数据风险、状态永久不收敛；
- Major：核心功能可继续但结果错误或相互矛盾；
- Minor：视觉、文字、可理解性或低频操作问题；
- Observation：环境缺失或主观现象，不直接判定代码失败。

## 7. 真实资源库读取

## 7.1 音声库 `E:\arsm`

1. 设置 → 资源库目录。
2. 点击“选择音声库目录”。
3. 选择 `E:\arsm`。
4. 确认页面只显示目录名称，不显示完整路径或内部授权值。
5. 点击“读取已有记录”。
6. 从点击开始计时。

必须验证：

- 按钮显示“读取中…”；
- 设置页状态显示正在读取；
- 顶栏同时显示正在读取；
- 此时切换首页和音声库，页面显示同一次读取状态；
- 不得一处显示“读取中”，另一处显示“已加载”；
- 120 秒内成功时显示完成统计；
- 超过 120 秒时显示明确超时、后台仍会完成、可以重试；
- 超时后按钮恢复可用；
- 后台晚到成功时，设置、顶栏、首页、音声库同时切换到完成状态；
- 失败时显示具体错误和重试入口；
- 不得永久停留在“读取中”。

记录：

```text
开始时间：
首个状态更新时间：
完成或超时时间：
作品数：
音轨数：
警告数：
设置/顶栏/首页/音声库状态一致：是 / 否
```

### 7.2 音乐库 `D:\CloudMusic\VipSongsDownload`
