# CURRENT ROADMAP — MVP-51

Version: `0.89.0-mvp51`

## 本轮目标

MVP-51 继续 Beta 0.1 后的小范围体验打磨，目标是让播放器大页更像“听音频的沉浸页”，而不是诊断或工程面板。

## 本轮范围

允许：

- 精修 `LyricsPanel` 的播放页焦点信息。
- 增加中文、媒体向的播放页提示。
- 优化无歌词 / 无字幕空状态。
- 新增一个小 service 承接展示判断。
- 更新文档和 verifier。

禁止：

- 不改扫描链路。
- 不改 `library-index.json` 写入 / 读取链路。
- 不改 HTMLAudio 播放内核。
- 不改字幕读取 IPC / 解析链路。
- 不接 SQLite。
- 不接下载器。
- 不接元数据抓取。
- 不接 mpv。
- 不删除、移动、重命名真实媒体文件。
- 不向 Renderer 暴露 `absolutePath` 或 `file://`。

## 下一步建议

MVP-52 可以做：

- 音声库 / 音乐库 Beta 回归后的小问题修复；或
- 播放器大页继续小范围视觉精修；或
- 手动打包版回归后的缺陷修复。

仍不建议直接进入 SQLite、下载器、元数据、mpv 或高级文件整理。
