# CURRENT ROADMAP — MVP-48

Current version: `0.86.0-mvp48`

## 本轮定位

MVP-48 是阶段性 **Beta 0.1 收口包**。

本轮只整理当前可用能力、回归验收重点、后置功能和交接文档，不改变真实扫描、索引写入、播放、字幕、打包或文件操作链路。

## 已进入 Beta 0.1 的能力

```text
选择目录
→ dry-run 扫描
→ 写入 / 备份 library-index.json
→ 读取 index 到音声库 / 音乐库
→ 本地音频播放
→ 字幕读取
→ 视频 / 图片 / 文件外部打开
→ 文件管理器定位
→ Windows 打包链路
→ 播放历史 / 继续播放
→ 队列与歌单持久化
→ 本地封面
→ 设置 / 诊断分层
→ 首页、资源库、播放器表层产品化
```

## 继续保持的硬边界

- UI 必须中文。
- 主界面媒体感优先，工程信息后置到诊断页。
- Renderer 不接收 absolutePath。
- Renderer 不接收 file://。
- 不删除、移动、重命名真实媒体文件。
- 不接 SQLite，除非明确进入后置阶段。
- 不接下载器、元数据抓取、mpv。

## MVP-49 建议

MVP-49 建议做播放器与首页的视觉继续打磨，或做 Beta 0.1 本机人工验收后的缺陷修复。

不要在 MVP-49 直接跳到 SQLite、下载器、元数据抓取或 mpv。
