# Codex / DeepSeek 提示词管理规则

从本轮开始：

1. 完整任务提示词必须放进源码包的 `AI_HANDOFF/` 目录。
2. 聊天中只给用户一句简短转发词。
3. 提示词必须写明基线、允许范围、禁止事项、命令、验收标准和报告格式。
4. 涉及实机测试或关键验收时，完整提示词必须在同一轮包内提供。
5. Codex 只用于必要的本机关键验收/Git；无额度时 DeepSeek 按同一文件执行。
6. 非必要不要安排 Codex。
7. AI 不得忽略源码包内的 `MVP130_EXPERIMENTAL_DO_NOT_MERGE.md`。

推荐短提示词格式：

```text
解压源码包，打开 AI_HANDOFF/<任务文件>.md，严格按文件执行；不要扩展范围，完成后只返回报告。
```
