# CURRENT ROADMAP — MVP-52

Version: `0.90.0-mvp52`

## 本轮目标

MVP-52 继续 Beta 0.1 后的小范围回归修复，重点修正音声库 / 音乐库浏览层的细节问题，让资源库页面更稳定、更中文、更少工程感。

## 本轮范围

允许：

- 轻量修复音声库空状态和筛选恢复提示。
- 轻量修复音乐库浏览摘要和曲目副标题显示。
- 新增一个小 service 承接浏览层回归文案和状态 chip。
- 在诊断页记录本轮回归修复内容。
- 更新文档和 verifier。

禁止：

- 不改扫描链路。
- 不改 `library-index.json` 写入 / 读取链路。
- 不改 HTMLAudio 播放内核。
- 不改字幕读取 IPC / 解析链路。
- 不接 SQLite。
- 不接下载器。
- 不接元数据抓取。
- 不接 mpv。
- 不删除、移动、重命名真实媒体文件。
- 不向 Renderer 暴露 `absolutePath` 或 `file://`。

## 下一步建议

MVP-53 可以继续做：

- 音声库 / 音乐库小范围视觉统一；或
- 播放器大页继续小范围视觉精修；或
- 手动打包版回归后的缺陷修复。

仍不建议直接进入 SQLite、下载器、元数据、mpv 或高级文件整理。
