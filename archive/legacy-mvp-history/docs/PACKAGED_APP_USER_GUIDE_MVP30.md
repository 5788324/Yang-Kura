# Yang-Kura 打包版使用说明 MVP-30

当前版本：`0.68.0-mvp30`

## 第一次打开后怎么用

1. 打开 `Yang Kura.exe` 或 portable exe。
2. 进入左侧「设置」。
3. 点击「选择音声库目录」或「选择音乐库目录」。
4. 选择你的本地资源目录，例如：

```text
C:\Users\YANG\Music\arsm.one
```

## 已经有 library-index.json

如果目录里已经有 `library-index.json`：

```text
设置
→ 打包版快速导入
→ 读取现有 index
→ 去音声库 / 音乐库查看
```

这一步不会重新扫描，也不会改动媒体文件。

## 第一次使用，或资源变更后

如果目录里没有 `library-index.json`，或者你新增了资源：

```text
设置
→ 打包版快速导入
→ 一键扫描并应用
```

软件会自动执行：

```text
dry-run 扫描
→ 生成写入预览
→ 写入 library-index.json
→ 如有旧 index 先备份
→ 读取并应用到页面
```

这一步只会写入或备份 `library-index.json`，不会删除、移动、重命名媒体文件。

## 播放与字幕

导入成功后：

```text
音声库 / 音乐库
→ 打开作品或歌曲
→ 点击播放
```

如果同名目录下存在 `.lrc` / `.srt` / `.vtt` / `.ass`，播放器会尝试读取并显示歌词 / 字幕。

## 视频、图片和文件

视频、图片、压缩包、文本等文件默认走系统外部打开。Yang-Kura 当前不内置视频播放器，也不记录外部播放器进度。

## 常见问题

### 打开后看不到真实资源

先进入「设置」，重新选择目录，然后点「读取现有 index」。

### 资源更新后页面没变

进入「设置」，点「一键扫描并应用」。

### 不想覆盖旧 index

写入时会自动备份旧的 `library-index.json`，生成类似：

```text
library-index.backup-时间戳.json
```

### 不要做什么

当前版本不提供删除、移动、重命名、批量整理真实媒体文件的功能。
