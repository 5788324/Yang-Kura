# Legacy MVP history archive

This directory preserves historical Yang-Kura material that is no longer part of the active release surface.

Contents:

- `root/`: historical `HANDOFF_MVP*`, `PACKAGE_MANIFEST_MVP*`, and old Codex prompts.
- `docs/`: MVP01–MVP111 planning and implementation notes plus fixture-era overview documents.
- `scripts/`: MVP01–MVP111 snapshot verifiers.

These files are retained for traceability. They are **not** current release gates and are not expected to run from their archived paths. Current validation is `npm run verify:stable` or its compatibility alias `npm run verify:all`.
